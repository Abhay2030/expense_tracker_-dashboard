# Smart Expense Tracker (Admin Dashboard)

A modern, interactive web application built with Python and Streamlit to track personal or business expenses. This project features a secure admin login, real-time data visualization, and comprehensive transaction management.

## Features
* Secure Admin Login portal.
* Interactive dashboard with automatic metric calculations (Monthly Income, Expense, Net Savings).
* Dynamic bar charts for expense categorization using Matplotlib.
* Simple, clean form for adding new income and expense transactions.
* Sortable and viewable transaction history.

## Tech Stack
* Python 3
* Streamlit (Frontend/UI)
* Pandas (Data Handling)
* Matplotlib (Data Visualization)

## Installation & Setup

1. Clone this repository:
   git clone https://github.com/YOUR_USERNAME/smart-expense-tracker.git

2. Navigate into the project directory:
   cd smart-expense-tracker

3. Install the required dependencies:
   pip install -r requirements.txt

4. Run the application:
   streamlit run app.py
